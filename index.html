О моих навыках

HTML 5

60%

Css 3 удивителен, svg, animation, css grid, flex-box, открывают большой простор для творчества с чистым css, каждый день узнаю что то новое и пробую применить.

Удобный фреймворк для ускорения верстки. Большинство проектов создаю с его применением. Он значительно упрощает и ускоряет работу.

JQuery
